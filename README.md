# GeoAnalyser

A tool for analysing how one GPS path is compared to another.  The tool can be used online here: [geo.aljoni.uk](//geo.aljoni.uk).

---

## How to Use

1. Select the "Ground Truth" straight line `.gpx` file
2. Select the tracked path
3. Click analyse
